# Facebook

- This is a clone app of facebook
- Need to login with google account to see or add posts
- Simple design of facebook homepage
- Hosted in Firebase

Will add more futures!

## Tech used: 

- HTML
- CSS (BEM convention)
- Materialize
- React
- Context API
- Firebase
